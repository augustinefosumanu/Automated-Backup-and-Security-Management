# Automated Backup and Security Management

<h2> Lab </h2>
In this lab, I developed and automated critical system tasks to enhance data security, integrity, and HIPAA compliance:<br/>

-  File Organization: Automated movement of patient, doctor, and treatment files using cron.<br />
-  Backup Automation: Scheduled full backups of critical directories and verified archive integrity.<br />
-  System Maintenance:<br />
  -  backup.sh – Backups /home directory.<br />
  -  update.sh – Automates system updates.<br />
  -  Cleanup.sh – Removes unnecessary files.<br />
-  Security & Compliance: Integrated scripts into system-wide cron jobs and scheduled Lynis security scans.<br />
